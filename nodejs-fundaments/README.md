Create the project package.json:
```sh
npm init -y
```

Start the project:
```sh
node src/server.js
```

Install [HTTPie](https://httpie.io/) to test HTTP from terminal:
```sh
brew install httpie
```

Run the server from the terminal:
```sh
http localhost:3333
```

Start the project with watch flag:
```sh
node --watch src/server.js
```
